# Java implementation of LXMF - Lightweight eXtensible Signal Transport for Reticulum

The reference implementation of [Reticulum network protocol](https://reticulum.network) of as well as extensions like [LXST](https://github.com/markqvist/LXST) are written in Python.

Just like Java implementation of the [reticulum-network-stack](https://github.com/sergst83/reticulum-network-stack) this is the Java version of the LXST extension for Reticulum that can be used with the reticulum-network-stack.

Seamless interoperability with the reference versions is paramount in both above of these Java implementations.
